第一种方法：用先打印正方形之后打印下三角 

```python
for i in range (1,10):
    line = ''
    for j in range (1,10):
        if i >=j:
            
            line += str (j) +'*'+str(i) +'='+ str(i*j) + ' '
    print(line)
```

第二种用format 格式化字符串

```python
for i range(1,10):
    line =''
    for j in range(1,10):
        line += "{}*{}={} ".foramt(j,i,(j*i))
    print(line)
```


第三种 j的取值范围是（1，i+1）

```python
for i in range(1,10):
    line =''
    for j in range(1,i+1):
         line += '{}*{}={} '.format(j,i,(i*j))
    print(line,end='\n')
```

```python
row 是行 column 是列
for i in range(1,10):
    for j in range(1,i+1):
        line = '{}*{}={} '.format(j,i,(i*j))
```

```python
       line = '{}*{}={} '.format(j,i,(i*j) ,end = endchar)
```

用列表方式来写 ,最后再解构

```python
for i in range(1,10):
    line = []
    for j in  range(1,i+1):
        line.append("{}*{}={} ".format(j,i,i*j))
    print(*line,sep=' ')
tip：再括号内换行不用换行符
for i in range(1,10):
    for j in range(1,i+1):
        print("{}*{}={}{}".format(j,i,j*i, '' if  j==2 and i<5 else ''),

用格式符对齐方式 写
for i in range(1,10):
    for  j in range(1,i+1):
        print("{}*{}={:<2}".format(j,i,i*j),
        end='\n' if  i == j else '\t')
```
